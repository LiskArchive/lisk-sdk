printSignedMessage

Example

Prints the signed message in the following format:

                                    -----BEGIN LISK SIGNED MESSAGE-----
                                    <message>
                                    -----BEGIN SIGNATURE-----
                                    <address>
                                    <signature>
                                    -----END LISK SIGNED MESSAGE-----

It is recommended in order to contain strcuture, in HTMl put the output into a pre tag.

signAndPrintMessage

Example

Signs a message with the secret and outputs the result for print.

It combines signMessageWithSecret and printSignedMessage

verifyMessageWithPublicKey

Example

Verifys a signed message with the public key. Either returns null if not valid, if valid returns the original message.
